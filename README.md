## Bookers
### 書籍管理ツール
ユーザー認証をして本の感想を投稿するアプリケーションです
投稿した本にはコメントをつけることができ、いいね機能も存在します
またSNSとして運用できるようフォローしたユーザー同士がチャットできる機能も存在します

### 実装機能
- ユーザー認証機能
  - ユーザー認証にはdeviseを使用
  - 画像投稿にはrefileを使用
- 感想投稿機能
- 投稿に対するコメント機能
- 投稿に対するいいね機能
- フォロー機能
- 検索機能
- ajax及びjqueryを用いたコメント・いいねの非同期通信の実装
- 過去一週間のいいねが多い順に結果を並び替え
- - ユーザー同士で140字までのDMのやり取りができるようにする

### 使用言語情報
  ruby 2.6.3
  rails 5.2.5
  bootstrap 4.5
  jquery 3.6.0
